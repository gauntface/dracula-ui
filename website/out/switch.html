Switch

Switch is an abstraction pattern that represents toggles. Use the Switch component as an alternative for Checkboxes or Radios when options are binary. Switches are implemented as Checkboxes under the hood, and work normally with forms.

Usage

Render a toggle component using checkbox under the hood.

  • Preview
  • HTML
  • React
<div class="drac-box">
  <input
    type="checkbox"
    name="demo"
    id="demo"
    class="drac-switch drac-checkbox drac-switch-purple"
    checked=""
  /><label for="demo" class="drac-text drac-text-white">Checked</label>
</div>
<Box>
  <Switch color="purple" id="demo" name="demo" defaultChecked={true} />
  <label htmlFor="demo" className="drac-text drac-text-white">
    Checked
  </label>
</Box>;

Disabled

A disabled switch is unusable and un-clickable.

  • Preview
  • HTML
  • React
<div class="drac-box">
  <div class="drac-box">
    <input
      type="checkbox"
      name="one"
      disabled=""
      id="one"
      class="drac-switch drac-checkbox drac-switch-white"
      checked=""
    /><label for="one" class="drac-text drac-text-white">Checked</label>
  </div>
  <div class="drac-box">
    <input
      type="checkbox"
      name="two"
      disabled=""
      id="two"
      class="drac-switch drac-checkbox drac-switch-white"
    /><label for="two" class="drac-text drac-text-white">Unchecked</label>
  </div>
</div>
<Box>
  <Box>
    <Switch
      id="one"
      disabled={true}
      defaultChecked={true}
      color="white"
      name="one"
    />
    <label htmlFor="one" className="drac-text drac-text-white">
      Checked
    </label>
  </Box>
  <Box>
    <Switch id="two" disabled={true} color="white" name="two" />
    <label htmlFor="two" className="drac-text drac-text-white">
      Unchecked
    </label>
  </Box>
</Box>;

Colors

Switches can be customized to use any of the Dracula UI theme colors.

  • Preview
  • HTML
  • React
<div class="drac-box">
  <input
    type="checkbox"
    name="colors"
    id="colors"
    class="drac-switch drac-checkbox drac-switch-cyan"
  /><label for="colors" class="drac-text drac-text-cyan">Checked</label>
</div>
<Box>
  <Switch color="cyan" id="colors" name="colors" />
  <label htmlFor="colors" className="drac-text drac-text-cyan">
    Checked
  </label>
</Box>;

Properties

name

The name of the form element.

string
color (required)

The Dracula UI theme color to be applied to the Switch.

"purpleCyan" | "yellowPink" | "cyanGreen" | "pinkPurple" | "animated" | "cyan" | "green" | "orange" | "pink" | "purple" | "red" | "yellow" | "white" | "black" | "blackSecondary" | "blackLight"
disabled

A disabled switch is unusable and un-clickable.

boolean
