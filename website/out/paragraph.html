Paragraph

Paragraph is a semantic component used for blocks of text with semantic meaning. Paragraph accepts all the same customization options as Text.

Usage

Show a paragraph within an interface.

  • Preview
  • HTML
  • React

basic

<p class="drac-text drac-line-height drac-text-white">basic</p>
<Paragraph>basic</Paragraph>;

Colors

Paragraphs can be customized to use any of the Dracula UI theme colors.

  • Preview
  • HTML
  • React

cyan

green

<div class="drac-box">
  <p class="drac-text drac-line-height drac-text-cyan">cyan</p>
  <p class="drac-text drac-line-height drac-text-green">green</p>
</div>
<Box>
  <Paragraph color="cyan">cyan</Paragraph>
  <Paragraph color="green">green</Paragraph>
</Box>;

Weights

Paragraphs can be defined in different font weights.

  • Preview
  • HTML
  • React

normal

semibold

bold

<div class="drac-box">
  <p class="drac-text drac-line-height drac-text-white">normal</p>
  <p class="drac-text drac-text-semibold drac-line-height drac-text-white">
    semibold
  </p>
  <p class="drac-text drac-text-bold drac-line-height drac-text-white">bold</p>
</div>
<Box>
  <Paragraph weight="normal">normal</Paragraph>
  <Paragraph weight="semibold">semibold</Paragraph>
  <Paragraph weight="bold">bold</Paragraph>
</Box>;

Properties

size

Controls the size of the text based on pre-configured Dracula UI sizes.

"md" | "lg" | "sm" | "xs"
weight

Controls the weight of the text.

"normal" | "semibold" | "bold"
lineHeight

Controls the line height of a text.

"md" | "lg" | "sm" | "xs" | "xl"
color

Controls the color of the text

"purpleCyan" | "yellowPink" | "cyanGreen" | "pinkPurple" | "animated" | "cyan" | "green" | "orange" | "pink" | "purple" | "red" | "yellow" | "white" | "black" | "blackSecondary" | "blackLight"
as

The HTML element to be used

"p" | "a" | "span"
