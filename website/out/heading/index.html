Heading

Heading is used to display headlines and other forms of hierarchical Text. Headings are similar to the base Text component, but restricted to certain sizes and font weight.

Usage

Heading is an abstraction/style that can/is applied to HTML heading elements.

  • Preview
  • HTML
  • React

Heading

<h2 class="drac-heading drac-heading-xl drac-text-white">Heading</h2>
<Heading>Heading</Heading>;

Colors

Headings can be customized to use any of the Dracula UI theme colors, including all solid, gradient, and animated colors.

  • Preview
  • HTML
  • React

pink

purpleCyan 

<div>
  <h2 class="drac-heading drac-heading-2xl drac-text-pink">pink</h2>
  <h2 class="drac-heading drac-heading-2xl drac-text-purple-cyan">
    purpleCyan
  </h2>
</div>
<div>
  <Heading size="2xl" color="pink">
    pink
  </Heading>
  <Heading size="2xl" color="purpleCyan">
    purpleCyan
  </Heading>
</div>;

Properties

size

The size and type of Heading to be used. Denotes hierarchy.

"2xl" | "xl" | "lg" | "md" | "sm" | "xs"
color

The Dracula UI color to be applied to the Heading.

"purpleCyan" | "yellowPink" | "cyanGreen" | "pinkPurple" | "animated" | "cyan" | "green" | "orange" | "pink" | "purple" | "red" | "yellow" | "white" | "black" | "blackSecondary" | "blackLight"
as

The HTML element to be used
